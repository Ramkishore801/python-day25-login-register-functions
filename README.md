# Day 25 - Python Login & Register System

## 📌 Concepts Used
- Functions in Python
- File Handling

## ⚙️ Features
- Register a new user
- Login with existing credentials
- Stores data in a file (users.txt)

## ▶️ How to Run
1. Install Python
2. Run the file:
   python main.py

## 🧠 Learning Outcome
Learned how to create and use functions, and how to store data using files.
